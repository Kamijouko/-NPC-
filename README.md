# 明日方舟 NPC 立绘分类数据集

本项目整理了明日方舟剧情 NPC 立绘，并按统一规则进行分类，提供两套分辨率：

- `Subjects-jpg`：600x600 预览尺寸 JPG
- `Subjects-fullres`：原分辨率图片（fullres）

## 分类规则

分类遵循以下原则：

1. 所有 NPC 立绘中的“通用立绘”归类到对应国家/地区目录。
2. 任何带有名字的独立 NPC，或疑似独立 NPC 的立绘，统一归入 `其它`。
3. 无法按国家/地区明确归类的立绘，归入 `？？？`。

示例结构如下：
